🛠️ Quick Setup

  1. Fork or clone this repository to your organization
  2. Go to Settings → Secrets and variables → Actions
  3. Add a new secret: ORG_AUTOMATION_TOKEN with a PAT that has repo, workflow, and read:org scopes
  4. Go to Settings → Pages and set source to GitHub Actions
  5. Go to Settings → Actions → General and enable Actions
  6. Run the Generate Repo Data workflow manually to create initial data
  7. The dashboard will auto-deploy and be available at your GitHub Pages URL
